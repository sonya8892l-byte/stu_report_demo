# 地图参谋 · 自适应脚手架

> L0自主；L1轻推（停顿30秒）；L2结构提示（首次无效尝试）；L3分步示范（两次无效尝试）；L4兜底（持续卡顿或挫败）。L4只完成当前操作，不越过课程解锁条件。

## 任务1：定坐标

| 等级 | AI回应策略 |
|---|---|
| L0 | 肯定其定位依据，并请学生继续记录来源。 |
| L1 | “先找指北针、经纬线或熟悉的河流名称，哪一个能帮你定方向？” |
| L2 | 提供四项核对框：方向、图例、河流名称、展项标题。 |
| L3 | 示范读取一处公开图例：“我先确认蓝线代表水系，再沿蓝线寻找名称；下一处由你来找。” |
| L4 | 打开可操作标注模板，让学生从现场照片中点选两个可见地名；无法辨认时标“待核”，不补史实路线。 |

## 任务2：布态势

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生解释一个标记会怎样影响行动。 |
| L1 | “你已经标了位置，再加一处河流或山地约束会怎样？” |
| L2 | 依次显示红军区域、封锁方向、候选渡口、地形约束四个空槽。 |
| L3 | 用与本课无关的示意地图演示“事实标记”和“推演标记”的不同样式。 |
| L4 | 根据学生已提交证据生成半成品底图，由学生确认位置并补三条理由；受保护路线仍保持锁定。 |

## 任务3：拟路线

| 等级 | AI回应策略 |
|---|---|
| L0 | 追问路线中最脆弱的一环。 |
| L1 | “从起点到终点之间，有没有过河后无法继续的断点？” |
| L2 | 显示目标、优势、风险、缺失信息四格模板。 |
| L3 | 用学生已标出的候选点示范如何比较一小段，不延伸到完整史实路线。 |
| L4 | 把学生已有点位连接成可编辑草线，要求本人选择保留或删除，并补充风险与未知；解锁前不显示答案。 |

## 年级调整
- 小学：使用图标和箭头，理由可语音提交。
- 初中：要求方向、图例和至少一项路线风险。
- 高中：增加交通线、补给与替代路线的条件比较。
