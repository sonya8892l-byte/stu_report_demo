# 通讯兵 · 自适应脚手架

> L0自主；L1轻推（停顿30秒）；L2结构提示（首次无效尝试）；L3分步示范（两次无效尝试）；L4兜底（持续卡顿或挫败）。脚手架始终保留史料与情境边界。

## 任务1：收残讯

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生再补一项角色无法知道的信息。 |
| L1 | “这件通信工具能传来什么？它有没有说明全局原因？” |
| L2 | 提供物件功能、可能收到、仍然未知、来源四格表。 |
| L3 | 从一条展项说明示范“材料明确说明”和“我推测”的写法。 |
| L4 | 提取照片中的高置信展项文字供学生确认，并生成已知/未知空白卡；不添加人物经历。 |

## 任务2：译行动

| 等级 | AI回应策略 |
|---|---|
| L0 | 让同伴复述并核对一处差异。 |
| L1 | “这张卡里，时间、地点、对象，有哪一项还不够清楚？” |
| L2 | 提供任务、时间地点、确认对象、缺失信息、上报条件五栏。 |
| L3 | 示范把一句复杂指令拆成动作和确认点，不补入卡片未写的信息。 |
| L4 | 将卡片原文切成可拖动短句，学生逐句归类；关键项缺失时必须选择“呼叫老师/请求确认”。 |

## 任务3：写边界

| 等级 | AI回应策略 |
|---|---|
| L0 | 检查每句话是否带来源类型。 |
| L1 | “这句话是角色当时知道的，还是你从全局地图后来看到的？” |
| L2 | 提供“史料依据/合理推断/未知”三种句子标签。 |
| L3 | 将一条越界句改写为带不确定性的课程写作示例，请学生处理下一句。 |
| L4 | AI只重排学生原句并标出缺来源处，学生补标签后提交；不生成历史人物直接引语。 |

## 年级调整
- 小学：口述60字左右，用颜色区分知道与不知道。
- 初中：完成80—150字记录和逐句来源标签。
- 高中：比较命令合法性、必要信息与组织信任边界。
