# 示形参谋 · 自适应脚手架

> L0自主；L1轻推（停顿30秒）；L2结构提示（首次无效尝试）；L3分步示范（两次无效尝试）；L4兜底（持续卡顿或挫败）。任何等级均受路线解锁规则约束。

## 任务1：排行动

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生为相邻两张卡说明先后依据。 |
| L1 | “先找日期；没有日期时，材料里有没有‘随后’‘再次’一类词？” |
| L2 | 将行动卡分为有明确日期、只有先后线索、待核三组。 |
| L3 | 示范如何用一张展项照片给一张已开放卡片标来源。 |
| L4 | 把已开放行动卡按证据线索排成可编辑序列，缺口保留锁形卡，由学生核对。 |

## 任务2：辨虚实

| 等级 | AI回应策略 |
|---|---|
| L0 | 追问另一种可能解释。 |
| L1 | “对方看到的信号，和我方真正需要的结果相同吗？” |
| L2 | 显示可见信号、敌方判断、真正目标、替代解释四栏。 |
| L3 | 用舞台灯光等非冲突类比解释“被看见的动作”和“实际目的”，再让学生回到史料。 |
| L4 | 把学生证据填入可见信号栏，其余栏只给句式框；未解锁的虚实结论不自动生成。 |

## 任务3：演反应

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生检查行动被识破时的备用方案。 |
| L1 | “至少再运行一次：让对方做出与你预期不同的反应。” |
| L2 | 提供“相信信号/识破信号”双分支推演框。 |
| L3 | 提醒逐步填写行动、反应、窗口和风险，先演一小步。 |
| L4 | 自动载入学生已有卡片并运行无结论预览；学生选择分支、解释结果后才能提交。 |

## 年级调整
- 小学：用“看到什么/猜什么/实际想做什么”三格图。
- 初中：至少运行两个反应分支并写失败风险。
- 高中：增加信号成本、窗口期和二阶调动分析。
