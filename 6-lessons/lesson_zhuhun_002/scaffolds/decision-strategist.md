# 决策参谋 · 自适应脚手架

> L0自主；L1轻推（停顿30秒）；L2结构提示（首次无效尝试）；L3分步示范（两次无效尝试）；L4兜底（持续卡顿或挫败）。脚手架不替小组投票。

## 任务1：列方案

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生为方案补一条反方理由。 |
| L1 | “这句话来自展项，还是你根据局势推出来的？” |
| L2 | 提供方案、背景、支持理由、反对理由、来源五栏表。 |
| L3 | 用学生的一条材料示范中性转述，避免添加直接引语。 |
| L4 | 将照片中已确认的展项摘要填入来源栏，学生选择对应方案并补正反理由；会议结论保持锁定。 |

## 任务2：比风险

| 等级 | AI回应策略 |
|---|---|
| L0 | 追问最不可逆的风险。 |
| L1 | “如果这个关键前提错了，失败代价能承受吗？” |
| L2 | 显示目标一致度、可靠度、收益、失败代价、可逆性五个评分轴。 |
| L3 | 示范为一个虚构方案评分并解释“分数需要证据”，不评价本课方案。 |
| L4 | 根据学生材料生成未评分矩阵，逐轴提出二选一问题；最终选择仍由小组确认，史实结果不出现。 |

## 任务3：复决策

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生指出一次判断变化的触发证据。 |
| L1 | “新材料改变了你的哪一个前提？” |
| L2 | 提供“新证据—保留—修改—行动责任”四段模板。 |
| L3 | 根据学生日志找到修改前后的两句话，请学生自行连接因果。 |
| L4 | 生成基于学生原文的对照稿，空出判断变化与责任人，由学生补齐并确认。 |

## 年级调整
- 小学：用“好处/危险/还能不能回头”三列卡片。
- 初中：完成五维矩阵并记录少数意见。
- 高中：增加前提概率、可逆性和决策复核节点。
