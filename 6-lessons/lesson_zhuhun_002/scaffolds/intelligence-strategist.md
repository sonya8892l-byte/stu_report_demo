# 情报参谋 · 自适应脚手架

> L0自主；L1轻推（停顿30秒）；L2结构提示（首次无效尝试）；L3分步示范（两次无效尝试）；L4兜底（持续卡顿或挫败）。L4只使用当前已开放知识。

## 任务1：读电文

| 等级 | AI回应策略 |
|---|---|
| L0 | 肯定准确字段，提醒为未知字段保留空白。 |
| L1 | “这段说明里，谁、何时、传了什么，哪一项最清楚？” |
| L2 | 显示发送者、接收者、时间、内容、来源五项提取框。 |
| L3 | 从学生照片中圈出一处清晰文字，示范转录并标OCR置信度。 |
| L4 | 自动填写高置信字段供学生逐项确认，其余统一标“未知/待人工核对”，不补全残文。 |

## 任务2：划盲区

| 等级 | AI回应策略 |
|---|---|
| L0 | 追问一项信息为何属于该象限。 |
| L1 | “你有证据证明对方知道，还是只能推测？” |
| L2 | 提供我方已知、敌方可能已知、双方未知、敌方可能误判四个投放区。 |
| L3 | 用一条学生证据示范“信息卡 + 来源 + 可靠度”，请学生处理下一条。 |
| L4 | 将已提交证据生成待分类卡片，学生逐张选择象限和可靠度；AI不替其确认敌方认知。 |

## 任务3：测判断

| 等级 | AI回应策略 |
|---|---|
| L0 | 请学生补充与当前判断相反的分支。 |
| L1 | “如果对方没有相信这个信号，会发生什么？” |
| L2 | 显示信号、判断A、判断B、利用窗口、失败风险五格模板。 |
| L3 | 用日常非冲突示例说明同一信号可能有两种解释，再返回历史证据。 |
| L4 | 从学生已选信号生成两个空白条件句，由学生填依据与风险；受保护行动链继续锁定。 |

## 年级调整
- 小学：四象限改为“知道/可能知道/不知道/可能猜错”图卡。
- 初中：每条信息需要来源和高、中、低可靠度。
- 高中：增加时效、来源独立性与反情报假设。
