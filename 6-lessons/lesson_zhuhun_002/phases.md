# 阶段编排

## Phase 1：局势入场
- 时长：25min
- 模式：集体（全班）
- 地点：集合教室或展馆指定教育空间
- 功能模块：A06(沉浸媒体), A01(文字输入)
- 触发条件：教师手动启动
- 结束条件：完成开场影片 + 提交初始判断

### 流程
1. 播放“3万对40万”局势导入影片或占位内容
2. 絮絮说明自己是AI学习同伴，本课以电子参谋员身份只按本轮可见情报协助推演
3. 学生阅读1935年1月遵义会议后的初始态势卡
4. 学生提交初始选择：继续北渡、原地作战、向东转移或暂不决策
5. AI只追问判断依据，不公布后续四渡路线
6. 教师开启角色领取

### 数据采集
- 初始方案
- 使用的证据条目
- 对兵力悬殊的第一反应
- 决策信心值（1-5）

### 转场
- 教师确认完成导入 → Phase 2

---

## Phase 2：展陈采证
- 时长：120min
- 模式：个人角色任务 + 同角色短协作
- 地点：中国共产党历史展览馆长征相关展区
- 功能模块：A01(多模态采集), A02(答题评测), A07(扫码), 位置导航
- 触发条件：Phase 1结束 + 教师确认
- 结束条件：教师手动推进或采证时间结束

### 流程
1. AI根据角色显示馆内动线与安全提示
2. 五种角色分别完成3个递进任务
3. 每条证据必须标记：展项/权威资料/课程材料/个人推测
4. 完成核心任务后获得对应战图图层
5. 时间银行分支任务可并行进行

### 安全与展馆规则
- 全程跟随教师和场馆允许动线，不跨越围栏、不触摸展品
- 拍摄前遵守展馆当日规定；禁止闪光灯时不得开启闪光
- 不允许拍摄其他参观者正脸或个人证件
- 与小组失联、身体不适或找不到集合点时立即呼叫老师
- 场馆临时闭展时，由教师切换到预置资料包，不引导学生绕行

### 转场
- 每组至少获得4层战图且教师确认 → Phase 3
- 未完成的角色保留已有证据，教师可开放资料补充卡

---

## Phase 3：四渡推演
- 时长：60min
- 模式：小组协作
- 地点：馆内教育空间或返程后的学习空间
- 功能模块：A03(拼合搭建), A04(兵棋推演), A05(讨论记录)
- 触发条件：Phase 2结束
- 结束条件：完成四轮决策与证据复盘

### 流程
1. 五层战图叠合，显示各角色证据但隐藏史实结果
2. 系统依次冻结在一渡、二渡、三渡、四渡前的时间点
3. 每轮只开放当时已经掌握的情报
4. 小组提交“目标—约束—方案—风险—预期敌方反应”
5. 系统再开放下一段史实，要求标记原判断保留或修正
6. 通讯兵记录：基层执行者实际能看到多少信息

### 关键原则
- 推演评价关注证据和推理过程，不以猜中史实路线为唯一标准
- 禁止用后续结果倒推“当时只有一个正确选择”
- AI协调角色发言，不代替小组投票或教师讨论

### 转场
- 五层战图齐全 + 四轮推演完成 → Phase 4
- 教师可在资料缺失时手动解锁，但系统记录缺失图层

---

## Phase 4：璇玑时刻
- 时长：30min
- 模式：小组讨论 → 全班交流
- 地点：学习空间
- 功能模块：A06(沉浸媒体), A04(双视角整合), A01(语音/文字)
- 触发条件：Phase 3完成或教师解锁
- 结束条件：完成双视角回应

### 流程
1. 战图从“全局指挥视角”切换为“基层士兵有限视角”
2. 展示“失散小战士追赶队伍”课程情境材料，并明确标注出处待核
3. 学生回答：看不到全局时，一个人凭什么判断、行动和坚持
4. AI要求区分史实证据、情境推断与价值判断
5. 小组形成双栏结论：战略层的行动逻辑 / 个体层的行动依据

### 认知转折
- 从“路线聪明”转向“组织如何形成可信行动”
- 从崇拜单一人物转向理解集体讨论、情报工作、组织执行和个体信念
- 对未核验情境保持证据边界

### 转场
- 教师组织分享后推进 Phase 5

---

## Phase 5：得意何在
- 时长：30min
- 模式：小组汇报 + 集体讲解
- 地点：学习空间
- 功能模块：A01(文字/语音), A05(小组对比)
- 触发条件：Phase 4结束
- 结束条件：每组形成带证据的核心判断

### 流程
1. 每组用3分钟回答核心问题
2. AI按“证据—判断—边界”整理各组差异，不做排名
3. 教师补充四渡赤水与遵义会议后独立自主、实事求是的历史意义
4. 回看初始方案，记录至少一次判断变化及触发它的证据
5. 完成迁移题：现实中何时应该坚持，何时应该调整方案

### 转场
- 教师推进 Phase 6

---

## Phase 6：归档与尾声
- 时长：10min
- 模式：个人
- 地点：学习空间
- 功能模块：学习报告
- 触发条件：Phase 5结束
- 结束条件：报告预览生成

### 流程
1. AI生成个人推演轨迹：初始判断、证据贡献、修正节点和最终结论
2. 学生确认哪些内容是史实、推断或价值表达
3. 絮絮告别并提醒：历史判断需要持续回到证据
4. 课程结束标记
