# 阶段编排

## Phase 1：沉浸叙事
- 时长：20min
- 模式：集体（全班）
- 地点：集合区域（午门广场或指定教室）
- 功能模块：A06(沉浸媒体)
- 触发条件：教师手动启动
- 结束条件：视频播放完成 + AI收集完初始猜想

### 流程
1. 播放"暴雨将至"沉浸短片（3min）
2. AI（絮絮）出场，自我介绍，建立关系
3. AI向每个学生提问：「你觉得故宫暴雨时会积水吗？为什么？」
4. 收集学生的初始假设（C类数据：C2认知数据）
5. 引出核心问题：「600年，为什么不积水？今天我们一起找答案」
6. 发放角色卡（扫码确认角色）

### 转场
- 教师确认所有学生完成扫码 → 推进到 Phase 2

---

## Phase 2：现场采证
- 时长：90min
- 模式：个人（按角色分散）
- 地点：故宫各区域（由角色决定）
- 功能模块：A01(多模态采集), A02(答题), A07(扫码)
- 触发条件：Phase 1 结束 + 教师确认
- 结束条件：教师手动推进 或 时间耗尽

### 流程
1. AI根据角色引导学生前往目标区域（位置导航卡片）
2. 每个角色执行3个递进任务（见 roles/*.md）
3. 任务完成后获得密符字母
4. 时间银行分支任务可并行进行

### 安全规则
- 安全围栏：超出故宫红墙范围 → AI警告 + 教师端预警
- 10分钟无操作 → AI主动问候确认状态
- 学生呼叫老师 → 教师端即时弹窗

### 转场
- 教师判断多数组完成 → 手动推进到 Phase 3
- 或全部组完成 → 自动提示教师可推进

---

## Phase 3：推理推演
- 时长：40min
- 模式：个人→小组过渡
- 地点：指定集合区域
- 功能模块：A01(文字输入), A03(拼合搭建), A04(沙盘推演)
- 触发条件：Phase 2 结束
- 结束条件：小组完成证据拼合

### 流程
1. AI引导学生整理采证阶段的发现
2. 每个角色撰写"我的发现报告"（文字+照片）
3. 小组汇合后，用拼合工具将6个角色的发现整合
4. 尝试还原完整的故宫排水路径

### 关键设计
- 任何单个角色都只看到系统的一部分，必须6人协作才能拼全
- AI在此阶段从"个人导师"切换为"小组协调者"
- 密符必须全部集齐才能进入 Phase 4

### 转场
- 全组密符集齐 Y-I-N-S-H-U → 解锁 Phase 4
- 教师可手动解锁（针对未全部完成的组）

---

## Phase 4：璇玑时刻
- 时长：30min
- 模式：小组协作
- 地点：集合区域
- 功能模块：A04(沙盘推演), A06(沉浸媒体)
- 触发条件：密符集齐 或 教师解锁
- 结束条件：暴雨模拟完成

### 流程
1. 小组在沙盘中搭建完整水系图（基于Phase 3的拼合结果）
2. 设置暴雨参数（降水量、持续时间）
3. 运行模拟——观察水流路径
4. AI揭示：「你们搭建的系统……能撑过暴雨吗？」
5. 播放暴雨动画验证

### 认知反转
- 学生以为自己已经理解了全部——但模拟结果可能暴露遗漏
- AI追问：「有什么环节你们可能忽略了？」
- 引导学生反思系统思维的盲区

### 转场
- 暴雨模拟完成 → 教师推进到 Phase 5

---

## Phase 5：总结汇报
- 时长：20min
- 模式：集体
- 地点：集合区域
- 功能模块：A01(文字输入/语音)
- 触发条件：Phase 4 结束
- 结束条件：教师推进

### 流程
1. 每组分享"我们组的发现"（2-3min/组）
2. AI辅助生成组间对比（不做排名，聚焦差异）
3. 回顾初始假设——「你最初的猜想对了吗？」
4. AI引导元认知反思：「如果再来一次，你会改变哪一步？」

### 转场
- 教师推进到 Phase 6

---

## Phase 6：尾声
- 时长：10min
- 模式：个人
- 功能模块：无（系统自动）
- 触发条件：Phase 5 结束

### 流程
1. AI生成个人学习报告预览（完整版后续推送）
2. 絮絮告别：「今天很开心陪你探索！下次见~」
3. 课程结束标记
