# 引河官 自适应脚手架

> 角色：引河官 🌊 — 负责追踪内金水河的路径、测量流速，并预测暴雨时的行为变化。

## 脚手架等级定义

| 等级 | 名称 | 触发条件 | 策略 |
|------|------|----------|------|
| L0 | 自主 | 学生进展顺利，无卡顿 | 仅给予肯定 |
| L1 | 轻推 | 停顿30秒以上或方向略偏 | 提出引导性问题 |
| L2 | 提示 | 尝试1次未果或明确表达困惑 | 给出具体方向提示 |
| L3 | 示范 | 尝试2次未果或出现挫败信号 | 展示部分示例或方法 |
| L4 | 兜底 | 长时间卡住或情绪低落 | 提供最大支持，确保学生能完成 |

**触发判断依据：** 停留时间、尝试次数、语言中的挫败信号（如"不会""太难了""不知道"）

## 任务脚手架

### 任务1: 追其源 — 追踪内金水河的路径

| 等级 | AI回应 |
|------|--------|
| L0 | "追踪得很准确！你已经找到河流的走向了。" |
| L1 | "水是往哪个方向流的？顺着水流方向走，能走到哪里？" |
| L2 | "从西北角开始跟踪——内金水河从那里进入故宫。" |
| L3 | "内金水河从故宫西北角流入，蜿蜒穿过太和门前的广场（这段呈弓形），然后从东南方向流出。你现在在哪一段？" |
| L4 | "内金水河完整路径：从西北角的入水口进入 → 沿西侧南流 → 在太和门前横穿广场（形成著名的弓形弯道，上有5座石桥）→ 继续向东 → 从东南角流出故宫 → 最终汇入护城河。试着在地图上标出来。" |

### 任务2: 测其流 — 测量水流速度

| 等级 | AI回应 |
|------|--------|
| L0 | "测量方法很有创意！科学家也是这样做的。" |
| L1 | "你能往水里放什么东西来计时？什么东西会跟着水流走？" |
| L2 | "一片树叶 + 手机计时器 = 流速测量工具。在两个标记点之间计时。" |
| L3 | "操作方法：1）在河边选两个相距约5米的标记点 2）把一片叶子放入上游标记处 3）计时到它到达下游标记 4）距离 ÷ 时间 = 速度。重复3次取平均值更准。" |
| L4 | "分步指南：标记A点和B点（间距约5米，用地砖接缝做标记）→ 同学1在A点放入树叶并喊'开始' → 同学2在B点按下计时器 → 叶子到达B点时停止计时 → 5米 ÷ 秒数 = 米/秒。做3次，把3个数加起来除以3。正常流速大约0.3-0.5米/秒。" |

### 任务3: 演其变 — 预测暴雨时的行为

| 等级 | AI回应 |
|------|--------|
| L0 | "预测很有道理！你考虑到了多种可能性。" |
| L1 | "水量突然变大时，河道能装得下吗？装不下会怎样？" |
| L2 | "内金水河平时只用了一部分容量。暴雨时水量可能增加10倍——河道还够用吗？" |
| L3 | "思考3个阶段：1）小雨——河道轻松应对 2）中雨——水位上升但不溢出 3）暴雨——水位接近满，其他排水系统需要帮忙分担。溢出的水会去哪里？" |
| L4 | "暴雨情景推演：平时水深约0.5米，河道深约2米 → 小雨：水深升至0.8米，无压力 → 中雨：水深升至1.5米，流速加快 → 暴雨：水深接近2米，此时两侧沟渠系统也在排水，减轻河道压力。如果极端暴雨超过河道容量，水会溢到广场上——这就是为什么广场也有自己的排水坡度。" |

## 年级调整

### 小学 (G3-G6)
- 追踪路径只需找到太和门前的弓形段即可
- 测流速可简化为"快还是慢"的定性判断
- 暴雨预测用"装水的杯子"类比
- 不要求精确数值计算

### 初中 (G7-G9)
- 要求画出河流路径简图
- 测流速需计算具体数值（米/秒）
- 暴雨预测需考虑至少2个阶段
- 鼓励思考"为什么河道不建成直线"

### 高中 (G10-G12)
- 期望分析弓形设计的水力学意义
- 测流速需多次测量并计算平均值和误差范围
- 暴雨预测需结合流量概念（流速 x 截面积）
- 鼓励讨论古代排水设计与现代标准的对比
