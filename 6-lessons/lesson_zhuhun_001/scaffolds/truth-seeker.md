# 真相官 自适应脚手架

> 角色：真相官 🔎 — 负责交叉验证各角色的证据、挑战假设、引入反证，并形成有边界的结论。

## 脚手架等级定义

| 等级 | 名称 | 触发条件 | 策略 |
|------|------|----------|------|
| L0 | 自主 | 学生进展顺利，无卡顿 | 仅给予肯定 |
| L1 | 轻推 | 停顿30秒以上或方向略偏 | 提出引导性问题 |
| L2 | 提示 | 尝试1次未果或明确表达困惑 | 给出具体方向提示 |
| L3 | 示范 | 尝试2次未果或出现挫败信号 | 展示部分示例或方法 |
| L4 | 兜底 | 长时间卡住或情绪低落 | 提供最大支持，确保学生能完成 |

**触发判断依据：** 停留时间、尝试次数、语言中的挫败信号（如"不会""太难了""不知道"）

## 任务脚手架

### 任务1: 汇其证 — 收集和整理各角色的证据

| 等级 | AI回应 |
|------|--------|
| L0 | "证据收集得很全面！你已经整合了多个角色的发现。" |
| L1 | "你已经问过所有5个角色了吗？还有谁的数据你还没收集到？" |
| L2 | "按证据可靠度分级：A级=亲眼所见亲手测量，B级=别人告诉你的，C级=查资料得到的，D级=猜测。" |
| L3 | "做一张证据卡片：每张写上——来源（哪个角色）、内容（发现了什么）、可靠度（A/B/C/D）、能证明什么。这样就能一目了然。" |
| L4 | "证据卡模板：[来源] 数龙官 / [证据] 太和殿有X个螭首 / [等级] A级-亲手数的 / [支持结论] 排水口数量足够覆盖整个台基。每个角色至少收集1-2张这样的卡片，然后按等级排序：A级的证据最可信。" |

### 任务2: 辨其伪 — 挑战假设，引入反证

| 等级 | AI回应 |
|------|--------|
| L0 | "批判性思维很棒！你没有轻易接受表面结论。" |
| L1 | "有没有什么证据和我们的'故宫排水完美'故事矛盾？" |
| L2 | "2023年北京暴雨时，故宫某些区域确实出现了积水——这说明什么？" |
| L3 | "对比两个事实：1）'故宫600年不积水'（网络说法） 2）2023年暴雨故宫部分区域积水（新闻报道）。哪个更可信？为什么？两者能同时成立吗？" |
| L4 | "引导分析2023年事件：原来的设计确实优秀（600年前的标准），但是——1）部分排水沟被后期修缮堵塞 2）现代暴雨强度可能超过古代设计标准 3）部分区域地面沉降改变了坡度。结论：古人的设计思想领先，但不等于'永远不会积水'。这就是'有条件的结论'。" |

### 任务3: 断其案 — 形成有边界的结论

| 等级 | AI回应 |
|------|--------|
| L0 | "结论很平衡！你既肯定了设计智慧，又指出了局限性。" |
| L1 | "你能说'故宫100%不会积水'吗？什么样的结论更科学？" |
| L2 | "'大约''在一般情况下''设计上能够'——这些词比'绝对''一定'更科学。" |
| L3 | "一个好的结论包含三部分：证据（我们发现了什么）+ 范围（在什么条件下成立）+ 局限（什么情况下可能不成立）。试试用这个结构写。" |
| L4 | "结论模板：'基于[我们测量的坡度、数过的螭首、追踪的沟渠]，我们认为[故宫排水系统设计精巧，在正常降雨条件下能有效排水]，但需要注意[极端暴雨可能超出设计容量，且部分设施因年久需要维护]。'用你自己的数据填进去。" |

## 年级调整

### 小学 (G3-G6)
- 证据分级简化为"自己看到的"vs"听说的"两级
- 不要求深入质疑，能提出一个疑问即可
- 结论用"我觉得……因为……"句式
- 2023年事件作为老师补充，不要求学生自行发现

### 初中 (G7-G9)
- 要求使用A/B/C/D四级证据分类
- 需要能识别至少一个"反证"
- 结论需包含"证据+结论"两部分
- 鼓励区分"设计意图"和"实际效果"

### 高中 (G10-G12)
- 期望独立发现证据之间的矛盾
- 要求分析网络流行说法的可信度
- 结论需包含完整的"证据+范围+局限"三段结构
- 鼓励讨论"科学结论"与"传播叙事"的区别
