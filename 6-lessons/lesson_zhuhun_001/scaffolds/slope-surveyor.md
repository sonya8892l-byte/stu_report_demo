# 测坡官 自适应脚手架

> 角色：测坡官 📐 — 负责观察地形坡度、设计测量方法，并理解坡度对排水的工程意义。

## 脚手架等级定义

| 等级 | 名称 | 触发条件 | 策略 |
|------|------|----------|------|
| L0 | 自主 | 学生进展顺利，无卡顿 | 仅给予肯定 |
| L1 | 轻推 | 停顿30秒以上或方向略偏 | 提出引导性问题 |
| L2 | 提示 | 尝试1次未果或明确表达困惑 | 给出具体方向提示 |
| L3 | 示范 | 尝试2次未果或出现挫败信号 | 展示部分示例或方法 |
| L4 | 兜底 | 长时间卡住或情绪低落 | 提供最大支持，确保学生能完成 |

**触发判断依据：** 停留时间、尝试次数、语言中的挫败信号（如"不会""太难了""不知道"）

## 任务脚手架

### 任务1: 察其势 — 观察水往哪里流

| 等级 | AI回应 |
|------|--------|
| L0 | "观察力很强！你已经注意到地面的倾斜趋势了。" |
| L1 | "下过雨后，水坑一般出现在哪边？这能告诉你什么？" |
| L2 | "故宫的地面北边比南边高。你站在太和殿广场上，感觉得到吗？" |
| L3 | "试试倒一点水在地面上，观察它往哪个方向流。这就是最简单的'找坡度'方法。" |
| L4 | "北高南低——这是故宫地形的秘密。水自然往南流，从后宫流向前朝，最终流出午门。你可以在地面上倒水验证。" |

### 任务2: 量其度 — 设计并执行坡度测量方法

| 等级 | AI回应 |
|------|--------|
| L0 | "方法很巧妙！你设计的测量方案很有创意。" |
| L1 | "身边有什么工具可以帮你测量高度差？想想水瓶、尺子、手机……" |
| L2 | "一个水瓶放在平面上 + 一个平整的板 = 简易水平仪。水面总是水平的！" |
| L3 | "教你做一个简易水平仪：把水瓶横放在一块平板上，看水面是否平行于板面。不平行说明有坡度。然后量出两端高度差。" |
| L4 | "坡度 = 高度差 ÷ 水平距离。操作步骤：1）找一段10米左右的路面 2）用水瓶水平仪确认方向 3）量出两端高度差 4）高度差 ÷ 距离 = 坡度百分比。故宫的坡度大约是2%。" |

### 任务3: 析其理 — 分析为什么故宫选择这个特定坡度

| 等级 | AI回应 |
|------|--------|
| L0 | "分析得很深入！你已经理解坡度设计背后的工程智慧了。" |
| L1 | "如果坡度更大或更小，会发生什么？想想极端情况。" |
| L2 | "太陡 = 水流太快，冲刷地面；太缓 = 水流不走，积水。2%是个工程平衡点。" |
| L3 | "2%坡度是一个工程最优解，原因有三：1）足够排水 2）不会冲刷 3）人走着不会觉得斜。这叫'不知不觉的坡度'。" |
| L4 | "想象一个滑梯：太陡滑太快（冲刷），太平滑不动（积水）。故宫的2%坡度像一个'刚刚好'的滑梯——水能流走，地面不被冲坏，走路也感觉不到斜。这就是600年前工匠的智慧。" |

## 年级调整

### 小学 (G3-G6)
- 不要求精确计算坡度百分比，感受"往哪边斜"即可
- 用"滑梯"类比代替工程术语
- 倒水观察即为完成测量任务
- 降低反思深度：能说出"北高南低"就好

### 初中 (G7-G9)
- 要求尝试计算坡度百分比
- 引入"高度差 ÷ 水平距离"公式
- 鼓励讨论坡度与排水效率的关系
- 可以使用手机水平仪辅助

### 高中 (G10-G12)
- 期望独立设计测量方案
- 要求计算坡度并与工程标准对比
- 鼓励讨论不同坡度的排水量差异（可引入简单流体力学概念）
- 减少脚手架：优先触发L0-L1，L3/L4仅在确实需要时给出
