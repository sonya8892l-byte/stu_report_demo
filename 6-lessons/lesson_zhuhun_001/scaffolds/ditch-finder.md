# 寻沟官 自适应脚手架

> 角色：寻沟官 🔍 — 负责发现、追踪和绘制故宫排水沟渠网络，理解层级化排水系统。

## 脚手架等级定义

| 等级 | 名称 | 触发条件 | 策略 |
|------|------|----------|------|
| L0 | 自主 | 学生进展顺利，无卡顿 | 仅给予肯定 |
| L1 | 轻推 | 停顿30秒以上或方向略偏 | 提出引导性问题 |
| L2 | 提示 | 尝试1次未果或明确表达困惑 | 给出具体方向提示 |
| L3 | 示范 | 尝试2次未果或出现挫败信号 | 展示部分示例或方法 |
| L4 | 兜底 | 长时间卡住或情绪低落 | 提供最大支持，确保学生能完成 |

**触发判断依据：** 停留时间、尝试次数、语言中的挫败信号（如"不会""太难了""不知道"）

## 任务脚手架

### 任务1: 寻其踪 — 发现可见的排水设施

| 等级 | AI回应 |
|------|--------|
| L0 | "发现得很好！你找到了重要的排水线索。" |
| L1 | "看看铺地砖的边缘，有没有特殊的缝隙或凹槽？" |
| L2 | "石板之间的缝隙就是线索——有些缝比其他的宽，那可能就是排水沟。" |
| L3 | "看到了吗？那些石头砌成的浅槽就是'明沟'。它们沿着建筑边缘延伸，把院子里的水引走。试着跟踪一条看它通向哪里。" |
| L4 | "故宫的排水沟叫'明沟'，就是明摆着的水沟。寻找方法：1）沿建筑台基边缘看 2）在院子的低处找 3）看石材拼接处的宽缝。找到后拍照记录位置。" |

### 任务2: 探其网 — 理解排水网络的层级结构

| 等级 | AI回应 |
|------|--------|
| L0 | "很好的发现！你已经看到沟渠之间的连接关系了。" |
| L1 | "这条小沟流向哪里？它最终汇入了什么？" |
| L2 | "小沟 → 中沟 → 大沟 → 河。就像小溪汇入小河，小河汇入大河。" |
| L3 | "试着像画树状图一样画出来：树叶=小院排水沟，树枝=干沟，树干=支沟，树根=内金水河。" |
| L4 | "故宫排水网络有4个层级：院落小沟（收集雨水）→ 干沟（汇集各院落）→ 支沟（连接各区域）→ 内金水河（总出口）。就像城市道路：小巷→街道→大路→高速。" |

### 任务3: 绘其图 — 绘制排水网络示意图

| 等级 | AI回应 |
|------|--------|
| L0 | "图画得很清晰！层级关系一目了然。" |
| L1 | "你的图里有没有遗漏的层级？数一数从最小到最大有几层。" |
| L2 | "试着在图上标出至少3个层级，用不同的线条粗细表示大小。" |
| L3 | "画法建议：最上面画各个院子（小圆圈）→ 用细线连到中间的干沟（中等线）→ 再连到下面的主沟（粗线）→ 最下面画内金水河。" |
| L4 | "模板参考：画3行。第1行：画3-4个小方框代表院落，每个框下面画向下的箭头。第2行：画一条水平线代表干沟，箭头都汇入这里。第3行：干沟向下的箭头指向'内金水河'。在每个层级旁标注名称。" |

## 年级调整

### 小学 (G3-G6)
- 只需找到2-3个排水沟即可完成任务1
- 层级理解简化为"小沟→大沟→河"三级
- 绘图可以是简笔画，不要求精确
- 用"水管游戏"类比沟渠网络

### 初中 (G7-G9)
- 要求找到不同类型的沟（明沟、暗沟）
- 理解完整4级层级结构
- 绘图需标注方向和层级关系
- 鼓励讨论为什么用层级而不是一条大沟

### 高中 (G10-G12)
- 期望发现明沟与暗沟的区别及设计原因
- 要求分析网络冗余性（某处堵塞时水怎么办）
- 绘图需接近系统拓扑图的精度
- 鼓励与现代城市排水管网对比
