# 弧形屏报告精简重构 · Writing Plan

## 一、目标

将 10 页完整报告精简为 **6 页纯内容页**（无封面、无封底），  
**撑满** 9984×2160 弧形屏，字大、图大、信息密度低。

---

## 二、尺寸计算

| 项目 | 当前值 | 改为 |
|------|--------|------|
| 总宽 | 9984px | 9984px（不变） |
| 总高 | 2160px | 2160px（不变） |
| 页数 | 10 页 × 940px | **6 页 × ~1552px** |
| 帧高 | 2016px | **2016px**（不变，上下留白 72px） |
| stage padding | 72px 64px | 72px 48px（左右缩窄） |
| gap | 64px | **48px** |

> 计算：(9984 − 48×2 − 48×5) / 6 = **1552px / 页**

每页从 940px 拉宽到 **1552px**，面积增大 65%，信息更宽松。

## 五、字体规划

| 层级 | 当前 | 改为 | 对标 |
|------|------|------|------|
| 页面主标题 `.frame-title` | 64px | **96px** | 与封面"故宫600年"同级 |
| 正文内容 | 24px | **32px** | 与现「你的这一天」数据卡文字同级 |
| 数据值（KPI） | 64px | **96px** | 醒目大数字 |
| 辅助说明 | 20px | **26px** | 比正文小一档，仍可远距阅读 |
| Badge 标签 | 20px | **26px** | 同辅助说明 |
| 页眉 kicker | 22px | **28px** | 适度放大 |

### 对应 CSS 变量（新增）
```css
:root{
  --fs-title: 96px;    /* 页面标题 */
  --fs-body: 32px;     /* 正文 */
  --fs-hint: 26px;     /* 辅助小字 */
  --fs-kpi: 96px;      /* 大数字 */
  --fs-badge: 26px;    /* 标签 */
  --fs-kicker: 28px;   /* 页眉 */
}
```

## 八、风险 & 注意

- 技能树 SVG 是 JS 动态生成的，需要调整 `viewBox` 和节点坐标以适配新宽度
- 雷达图的点击交互需要重新绑定到 P5/P6 的新 ID
- 1552px 宽页面下，原本竖向排列的卡片可以改为横向，更好利用空间
- 字体只用系统字体（宋体 + 苹方），无需额外加载
